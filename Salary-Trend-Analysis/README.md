# Salary Trends Across Data Careers | Power BI Dashboard

## Project Overview
This project explores **salary trends in data-related roles** such as Data Analyst, Data Scientist, and Data Engineer for the year 2024.  
It focuses on how **experience level, company size, and country** affect salaries in the data industry.

The dashboard was built completely in **Power BI**, including **data cleaning, DAX measures, and visualizations**.

---

## Objectives
- Analyze salary distribution across roles and countries.  
- Identify the most in-demand and highest-paying positions.  
- Understand how experience level and company size impact salaries.  
- Visualize salary growth over time.

---

## Insights
- **Most Demanded Role:** Data Scientist  
- **Highest Salary:** 600K  
- **Country Differences:** Noticeable variation between regions  
- **Annual Salary Growth:** ~12.5%  
- **Experience Impact:** Executives earn up to 3x more than entry-level roles
